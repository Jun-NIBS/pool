This folder contains code, data, plots and images for normalizing peptide activity across training sets to the same scale, and ranking good hits based on normalized values.

Normalization:
    norm_util.R
    norm_script.R
    norm_test.R (unit test)

Ranking good hits:
    pickhit_util.R
    pickhit_script.R
        This code generates tables of ranking.
    imageJ_script.py
        This code should be copy pasted into imageJ to generate images of ranking.
