# R files
generate_recom_set.R:
  Generate a recommendation set whose rows are features of peptides.

local_search.R:
  Perform local improvement search.

plot.R:
  Plot graphs for probability improvement data.

prob_improv_of_set.R:
  Compute probability improvement of a set.


# Data
prob_improv_S.csv:
  Probability improvement data generated by local_search.R

prob_improv_S_uniq.csv:
  Same as above, except that the version of local_search.R
  that outputs this tests uniqueness of each row of the
  recommendation set while updating the set.
